# 网页关键词搜索工具

抓取指定网页并在论坛页面中实现翻页搜索功能。

## 功能特点

🔍 搜索功能

- 多页面搜索：自动跟踪分页链接和内容链接

- 关键词匹配：支持区分大小写的搜索选项

- 上下文显示：显示关键词匹配的上下文内容

🌐 网页处理

- 智能分页检测：自动识别各种分页结构

- 域名限制：只在同一域名内搜索，避免爬取外部链接

- 连接管理：使用会话保持和超时控制

💻 用户界面

- 直观的Tkinter界面：易于使用的GUI

- 实时进度显示：搜索过程中显示当前状态

- 滚动文本框：支持大量结果的显示

## 安装依赖

在运行程序前，需要安装以下Python库：

```bash
pip install requests beautifulsoup4 lxml
```

## 使用方法

1. 输入目标网址：例如论坛或网站的URL

2. 输入关键词：要搜索的关键词或短语

3. 设置搜索深度：控制搜索的页面层级（1-10层）

4. 点击开始搜索：程序会自动抓取页面并查找关键词

## 注意事项

- 请遵守网站的robots.txt规则和爬虫政策

- 避免对服务器造成过大压力，可调整搜索深度限制

- 某些动态加载的网站可能需要使用Selenium等工具

这个工具特别适合论坛内容搜索，能够自动处理分页并全面搜索整个论坛的相关内容。你可以根据需要进一步扩展功能，比如添加延时控制、结果导出等。
