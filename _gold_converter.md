# 金价转换程序功能说明

## 1. 主要功能

- **双向转换：** 支持国际金价（美元/盎司）与国内基础金价（人民币/克）相互转换

- **实时汇率：** 从多个汇率API源自动获取美元兑人民币汇率

- **手动调整：** 可手动输入或调整汇率值

- **计算透明：** 显示详细计算过程和公式

## 2. 核心计算公式

```python
# 国际转国内
国内金价 = 国际金价 ÷ 31.1035 × 汇率

# 国内转国际  
国际金价 = 国内金价 × 31.1035 ÷ 汇率
```

## 3. 使用说明

**安装依赖：**

```bash
pip install requests
```

**运行程序：**

```bash
python gold_converter.py
```

**操作步骤：**

1.程序启动后自动尝试获取实时汇率

2.在"国际金价"框输入美元/盎司价格，点击"→ 转国内价"

3.或在"国内基础金价"框输入元/克价格，点击"→ 转国际价"

4.点击"获取实时汇率"更新最新汇率

5.汇率框可直接手动修改数值

## 4. 汇率获取策略

程序使用多个备用API源：

- exchangerate-api.com（主要）

- open.er-api.com（备用）

- currencyapi.com（备用）

如果所有API都不可用，程序会使用模拟数据确保基本功能正常。
