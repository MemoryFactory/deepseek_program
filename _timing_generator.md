# 时序图生成程序具有以下功能：

## 主要特性

1. 图形界面：使用tkinter创建用户友好的界面

2. Excel文件读取：读取包含真值表的Excel文件

3. 时序图生成：

    - 0画低水平线，1画高水平线

    - 值变化时画垂直线连接

    - 所有信号按时间对齐

4. 参数可调：

    - 信号间距

    - 时间单位宽度

    - 线宽

5. 文件导出：

    - JPG格式图片

    - Excel格式（使用边框表示时序关系）

## 使用说明

1. 运行程序后，点击"选择Excel文件"加载数据

2. Excel格式要求：

    - 第一列：信号名称

    - 后续列：0或1组成的时序数据

3. 调整参数后点击"生成时序图"

4. 可以保存为JPG图片或带边框的Excel文件

## Excel边框规则

- 上边框粗线：表示值为1

- 下边框粗线：表示值为0

- 侧边框粗线：表示值发生变化

## 用途

> 可视化数字电路的时序关系，适用于数字逻辑设计和时序分析。
